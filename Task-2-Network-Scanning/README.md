# Task-2: Network Scanning & Vulnerability Assessment

## Cybersecurity Internship – ApexPlanet

This repository contains the documentation and analysis of network scanning, vulnerability assessment, and packet analysis performed in a controlled virtual lab environment.

---

## Objective

The objective of this task is to identify active hosts, detect open ports and services, perform vulnerability assessment, and analyze network traffic using professional cybersecurity tools.

---

## Lab Environment

Attacker Machine:
- Kali Linux

Target Machine:
- Metasploitable2 / DVWA

Virtualization Platform:
- Oracle VirtualBox

Network Configuration:
- Host-Only Adapter Network

---

## Tools Used

- Nmap – Network scanning and service detection
- Nessus Essentials – Vulnerability assessment
- Wireshark – Network traffic analysis
- netdiscover – Host discovery
- whois – Information gathering

---

## Task Activities

- Discovered target system using netdiscover
- Verified connectivity using ping
- Performed information gathering using WHOIS
- Scanned open ports and services using Nmap
- Performed vulnerability assessment using Nessus Essentials
- Captured and analyzed HTTP traffic using Wireshark

---

## Key Findings

- Multiple open ports detected
- Running services identified
- Critical vulnerability including backdoor detected
- HTTP login traffic captured and analyzed
